Phase 4: Testing and Debugging

I have ran netcat for this server a lot to try and understand the server. The most common errors were:
  • Incorrect headers
  • Downloading the file when I wanted it to display
  • The page not loading because of incorrect bytes for the headers
If you would like to test that the server operates correctly, I would recommend starting it and using netcat. If you go to your terminal while the server is running, just run "nc localhost 8000". Then type in "GET /index.html HTTP/1.1". Then press enter twice and you can see your headers for that page and the html of the page.
