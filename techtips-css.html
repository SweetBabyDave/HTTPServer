HTML Elements

<html>

  • Purpose: The HTML root element. All elements must be a descendant of this element
  • Display: Neither
  • Parents: None

<h1>

  • Purpose: The first of 6 different levels of section headings. It helps the html document to flow better.
  • Display: Block-level
  • Parents: Any element that accepts flow content

<p>

  • Purpose: The HTML paragraph element. It is used to structurally group related content.
  • Display: Block-level
  • Parents: Any element that accepts flow content.

<body>

  • Purpose: Represents the content of the HTML document.
  • Display: Block-level
  • Parents: Must be second element of <html>

HTML Character Entities

&gt; (Greater Than)

  • Purpose: The way to show the greater than symbol in an HTML document
  • Symbol: >

&lt; (Less Than)

  • Purpose: The way to show the less than symbol in an HTML document
  • Symbol: <

&amp; (Ampersand)

  • Purpose: The way to show the ampersand symbol in an HTML document
  • Symbol: &

CSS Selectors

element

  • Purpose: Matches elements by node name. Letting you be able to select all elements of the same type.
  • How To Use: In the header or CSS file, write the element name then give it style properties.

.class

  • Purpose: Matches the elements based on contents set by the user in the class.
  • How To Use: You assign a class by doing it in the header or the separate CSS file. You just write it as .[insert class name here] then you can make class adjustments.

#id

  • Purpose: Matches an element based on the set value by the user for the id attribute.
  • How To Use: You assign an id in the header or separate CSS file. You just write #[insert name of id here] and set the style properties.

:nth-child

  • Purpose: Selects elements based on their positioning in a group.
  • How To Use: You assign an nth-child in the header or separate CSS file. You write :nth-child(which children you want different) then set the style properties.

CSS Properties

background-color:

  • Purpose: A way to select and change the color of the block that the element is referring to.
  • Possible Value: lemonchiffon

color:

  • Purpose: A way to select and change the color of the inline that the element is referring to.
  • Possible Value: darkred

font-style:

  • Purpose: A way to select and change the way that the font is seen by the user.
  • Possible Value: italic
